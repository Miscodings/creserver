Introduction

Alrighty. Resource packs. Almost every Minecraft server nowadays uses one. Plugins like ModelEngine REQUIRE them.

This is actually a relatively large topic. There's a lot of nuances regarding resource packs, but they really are not at all as complicated as they may seem.

For the purposes of this app, I'm going to assume that you're using a plugin to manage your resource pack for you (something like Nexo or maybe even MythicCrucible. Please do NOT use ItemsAdder!). But although I'll be using Nexo's commands and interface, the topics you'll be learning are applicable to any type of resource pack, regardless of how it was put together. All Minecraft resource packs behave the same way.
